# Kata

## Count Of Numbers

Count how often each number occurs in a given list of numbers and return it.

Extras:
- Use RestController
- Write a test

Example:

| Given  | [1,2,3,4,5,1,2]                                                                       | []  | [3,2,1,2]                                      |
|--------|---------------------------------------------------------------------------------------|-----|------------------------------------------------|
| Result | {<br />"1": 2,<br />"2": 2,<br />"3": 2,<br />"4": 1,<br />"5": 1,<br />"6": 1<br />} | {}  | {<br />"1": 1,<br />"2": 2,<br />"3": 1<br />} |
